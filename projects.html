Disgaea Clickerclose

Anyone who knows my video game habits knows that I've put countless hours into the tactical role-playing game series Disgaea, which is more of a challenge to increase character statistics to otherwise unseen levels than anything else. I felt that a clicker game loosely based on the series could hold value as a fan work because the main gameplay challenge of Disgaea is to incrementally increase numbers, almost solely to watch them go up. I built Disgaea Clicker using Bootstrap and JavaScript with the focus of designing a good interface for playing the game on mobile devices, as many browser-based clicker games don't render very well outside of desktop environments.

Fire Emblem Citadels

Customized Board Game Printing - Fire Emblem Citadelsclose

As an avid table-top game player, Citadels has been one of my favourites for over a decade. After an unfortunate accidental Diet Coke spill, I decided that I should buy a new set, as my existing copy of the game was already quite worn. Realizing that I was not a great fan of the game's visual aesthetics, I decided to design a new set of cards using assets from the (then) newly released Fire Emblem: Heroes. After spending hours in Adobe Photoshop, I sent the files off to Print & Play who then sent me a one-of-a-kind physical copy of "Fire Emblem Citadels". The game and rules remain unchanged; the only editing I did was to rulebook text that was strangely gender specific (the player was always refered to with masculine pronouns) by making these statements gender-neutral. Friends who've previously played Citadels have had no problem playing Fire Emblem Citadels.

Single Player Edition

Tales of the Arabian Nights - Single Player Editionclose

Using RenPy, a popular Python visual novel scripting engine, I'm digitizing the board game Tales of the Arabian Nights. The original table-top board game is heavily storytelling-driven with most player decisions being made through obfiscated reaction matrices. Since the game has very little player interaction, group play is stimulated by players taking turns reading from the "Book of Tales" (a large tome the game comes with that details all of the possible outcomes of a player's decisions). By automating the only social aspect of the game, the need for more than one player for the game to be enjoyable disappears.

Made in Waterloo by Steven Ye using Materialize.
